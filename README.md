rockscissorspaper
=================

Leap motion rock,scissors,paper recognition module from HackerNest@toronto

Using:
1. Include leap.js
2. Init module
	RockScissorsPaper.init();
3. Start tracking 
	RockScissorsPaper.startTracking(function(f) {console.log(f);});	
4. Stop tracking 
	RockScissorsPaper.stopTracking();
